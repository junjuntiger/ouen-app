## 手順4：Firebase の設定

### 4-1. Authentication の有効化

1. [Firebase Console](https://console.firebase.google.com/) を開く
2. `owenweb-54f51` プロジェクトを選択
3. 左メニュー → **Authentication** → **始める**
4. **Sign-in method** タブ → **電話** を有効化
5. **テスト用電話番号** を追加（開発中はこれを使う）
   - 例：+81 90-0000-0001 / コード: 123456

### 4-2. Firestore の有効化

1. 左メニュー → **Firestore Database** → **データベースを作成**
2. **本番環境モード** で作成
3. リージョン：`asia-northeast1`（東京）を選択
4. **ルール** タブで以下を設定する：

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && request.auth.uid == userId;
    }
    match /transactions/{transactionId} {
      allow read: if request.auth != null &&
        (resource.data.fromUserId == request.auth.uid ||
         resource.data.toUserId == request.auth.uid);
      allow create: if request.auth != null &&
        request.resource.data.fromUserId == request.auth.uid &&
        request.resource.data.paid >= request.resource.data.price;
      allow update: if request.auth != null &&
        resource.data.toUserId == request.auth.uid &&
        resource.data.status == "pending";
    }
  }
}
```

## 手順6：Vercel へのデプロイ

### 6-1. Vercel にプロジェクトを登録

1. [Vercel](https://vercel.com/) にログイン
2. **New Project** → GitHubリポジトリ `OUEN-APP` を選択
3. **Framework Preset**：Vite を選択

### 6-2. 環境変数を Vercel に設定

Vercel プロジェクト設定 → **Environment Variables** で  
`.env.local` の内容をすべて登録する。

### 6-3. デプロイ実行

```bash
git add .
git commit -m "初回コミット"
git push origin main
```

GitHub への push で Vercel が自動的にデプロイを開始する。

## よくある問題と対処法

### SMS認証で「送信に失敗しました」

**原因：** reCAPTCHA の初期化タイミングの問題

**対処：**
1. Firebase Console → Authentication → Sign-in method → 電話
2. テスト用電話番号を追加して開発中はそちらを使う
3. `localhost` が Firebase の承認済みドメインに含まれているか確認

---

### `Cannot find module 'firebase/app'` エラー

**原因：** npm install が完了していない

**対処：** node_modules フォルダを削除してから `npm install` を再実行する

---

### Firestore の権限エラー（`PERMISSION_DENIED`）

**原因：** Firestoreセキュリティルールが正しく設定されていない

**対処：**
1. Firebase Console → Firestore → ルール を確認
2. 手順4-2のルールが正しく設定されているか確認

---

### Vercel デプロイ後に白画面

**原因：** 環境変数が設定されていない

**対処：**
1. Vercel → プロジェクト → Settings → Environment Variables
2. `VITE_FIREBASE_*` がすべて設定されているか確認
3. 再デプロイ：Vercel → Deployments → Redeploy

---

### `FIREBASE_PRIVATE_KEY` の改行エラー

**原因：** Vercelの環境変数で改行が正しく認識されない

**対処：**
Vercelの環境変数設定画面で値を入力するときは `\n` を含めたまま  
一行の文字列として入力する（自動でエスケープ解除される）。
